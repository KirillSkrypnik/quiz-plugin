== Description ==
The Quiz plugin is a tool that allows you to easily create and conduct surveys on your website or app. With this plugin, you will be able to get valuable opinions and feedback from your users.
Create a variety of surveys on any topic, ask questions with a choice of one or more answer options. Customize available response options. Gain valuable insights into your audience's opinions and preferences to make informed decisions and improve your product or service. Don't miss the opportunity to interact with your audience and gain valuable opinions using the Poll plugin. Create surveys, collect data, and make informed decisions based on the opinions of your audience.

== Installation ==
This section describes how to install the plugin and get it working.
1. Upload `Quiz Imedia Team` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
